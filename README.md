# Brainfuck interpreter

Интерпретатор языка Brainfuck, написанный на Haskell. Поддерживает гибкую настройку параметров виртуальной машины через аргументы командной строки.

## Что такое Brainfuck?

Brainfuck — минималистичный язык программирования. Его виртуальная машина состоит из массива ячеек памяти и указателя текущей ячейки — по аналогии с лентой и головкой [машины Тьюринга](https://ru.wikipedia.org/wiki/%D0%9C%D0%B0%D1%88%D0%B8%D0%BD%D0%B0_%D0%A2%D1%8C%D1%8E%D1%80%D0%B8%D0%BD%D0%B3%D0%B0). Ввод и вывод данных осуществляется через команды `,` и `.`.

Язык состоит всего из восьми команд:

| Команда | Действие |
|---------|----------|
| `>` | Переход к следующей ячейке |
| `<` | Переход к предыдущей ячейке |
| `+` | Увеличить значение текущей ячейки на 1 |
| `-` | Уменьшить значение текущей ячейки на 1 |
| `.` | Вывести значение текущей ячейки |
| `,` | Считать значение в текущую ячейку |
| `[` | Начало цикла (перейти к `]` если текущая ячейка равна 0) |
| `]` | Конец цикла (вернуться к `[` если текущая ячейка не равна 0) |

Все остальные символы в исходном коде игнорируются и могут использоваться как комментарии.

В классическом Brainfuck: 30 000 ячеек, размер ячейки — 1 байт (значения 0–255 по модулю 256).

## Особенности реализации

- Вывод (`.`) показывает числовое значение ячейки **и** соответствующий ASCII-символ: `Out: 72 (H)`
- Ввод (`,`) принимает как **число** (например, `72`), так и **символ** (например, `H`) — в последнем случае берётся ASCII-код первого символа строки
- При старте выводится отладочная информация: размер памяти, режим зацикливания и модуль ячейки
- Принимаются файлы с расширениями `.bf` и `.b`

## Сборка

Для сборки необходим компилятор [GHC](https://www.haskell.org/ghc/).

```bash
ghc -o bf src/bf_main.hs src/bf_utils.hs
```

В репозитории уже есть собранный бинарный файл `app/bf.exe`.

## Использование

```
bf <файл> [флаги]
```

### Флаги

| Флаг | Значение по умолчанию | Описание |
|------|-----------------------|----------|
| `-s <n>` | `30000` | Размер банка памяти (количество ячеек) |
| `-l` | выключено | Зациклить массив памяти (указатель переходит с конца на начало и обратно) |
| `-c <n>` | без ограничений | Размер ячейки: значения хранятся по модулю `n` |

### Примеры запуска

```bash
# Классический Brainfuck (30 000 ячеек, байтовые ячейки)
bf example/hello.bf -s 30000 -c 256

# Зациклить память, 100 ячеек
bf program.bf -s 100 -l

# Неограниченные целые числа в ячейках
bf program.bf

# Комбинация флагов
bf program.bf -s 1000 -l -c 256
```

## Примеры программ

В папке `example/` находятся готовые программы:

| Файл | Описание |
|------|----------|
| `hello.bf` | Hello World (через цикл умножения) |
| `hello1.bf` | Hello World (прямым заданием значений) |
| `echo.bf` | Эхо: повторяет введённые символы |
| `sum.bf` | Сумма двух чисел (вводятся с клавиатуры) |
| `product.b` | Произведение двух чисел |
| `fact.bf` | Факториал числа |
| `pow2.bf` | Степень двойки |
| `mod.bf` | Остаток от деления |
